Impatient Vim

(^ = Touche Ctrl)

Sauvegarder/Quitter

  • Sauvegarder : :w
  • Quitter: maj Z Q
  • Quitter et Sauvegarder : maj Z Z

Mode Insertion

  • Mode Insertion : I
  • Sortir du mode Insertion : ^C
  • Mode Insertion, debut de ligne : maj I
  • Mode Insertion, fin de ligne : maj A

Selections

  • Commencer Selection : maj V
  • Selection zone : ^V
  • Supprimer Selection : Selection + D
  • Copier Selection : Selection + Y

Edition de Ligne

  • Supprimer une Ligne : D D
  • Copier une Ligne : Y Y
  • Coller une ligne dessous : P
  • Coller une ligne dessus : maj P
  • Creer une nouvelle ligne dessous : O
  • Creer une nouvelle ligne dessus : maj O
  • Remplacer : maj R

Divers

  • Annuler : U
  • Refaire : ^R

Encore plus...

  • Pour apprendre a utiliser VIM, taper vimtutor dans un terminal.
